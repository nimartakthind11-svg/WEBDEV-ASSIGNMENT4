This is a basic JavaScript quiz that uses popup prompts to ask General Knowledge questions. After the user enters their response, the software checks if it is accurate. The final score is displayed at the end.

It is simple to test and learn fundamental JavaScript concepts like arrays, functions, loops, and condition checks because the quiz operates directly in the browser console.

To run the test:

 1.open your browser. 
 2.Press Shift + Ctrl + I.
3. Click on the Console tab.
4. Copy the quiz code.
5. To begin, press Enter.
